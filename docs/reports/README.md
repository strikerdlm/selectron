v0.6 audit containment work and may use retired fixture names, old manuscript
framing, and historical validation language.

Do not use these reports as current claim sources for:

- validated crew selection;
- medical clearance;
- empirically calibrated analog pEVAC, pLOCL, or CHI;
- NASA-equivalent HSRB verdicts;
- operational planning recommendations;
- external analog-outcome validation.

The current claim boundary is:

- Stage A is an uncertain-weight MCDA demonstration, not a suitability
  posterior or select/reject instrument.
- Stage B outputs are conditional scenario distributions under loaded priors
  and operator assumptions, not calibrated absolute analog-risk forecasts.
- K15 is an inter-model verification benchmark, not external validation.
- Valid accepted active-parameter evidence coverage is 0/4,849.
- Communications-delay and other profile effects are proposal/exploratory
